# Домашнее задание №2

## Задачи

### A. Заправки

В стране есть $N$ городов, некоторые из которых соединены двусторонними дорогами.

Для проезда по одной дороге требуется один бак бензина. Стоимость бака зависит от города, в котором он покупается. Покупать бензин заранее нельзя.

Требуется добраться из города $1$ в город $N$, потратив минимальную сумму.

Ограничения:

```math
1 \leq N \leq 100
```

Стоимость бензина в каждом городе:

```math
0 \leq c_i \leq 100
```

Между двумя городами существует не более одной дороги. Петли отсутствуют.

**Ввод:** число $N$, стоимости бензина в городах, число дорог $M$, затем пары городов, соединённых дорогами.

**Вывод:** минимальная стоимость пути из города $1$ в город $N$ или `-1`, если пути не существует.

### B. Алхимики Средневековья

Дан набор алхимических реакций. Каждая реакция позволяет превратить одно вещество в другое.

Реакции являются направленными: возможность превратить вещество $A$ в вещество $B$ не означает, что возможно обратное превращение.

Требуется определить минимальное количество реакций, необходимых для получения заданного вещества из исходного.

Ограничения:

```math
0 \leq m \leq 1000
```

Во входных данных упоминается не более $100$ различных веществ. Название вещества содержит не более $20$ латинских букв.

**Ввод:** число реакций $m$, затем реакции в формате `вещество1 -> вещество2`. После них записаны исходное и требуемое вещества.

**Вывод:** минимальное количество реакций или `-1`, если требуемое вещество получить невозможно.

### C. Выход из лабиринта

Дан прямоугольный лабиринт размером $N \times M$.

Свободная клетка обозначается символом `0`, а клетка, занятая деревьями, — символом `1`.

Из клетки разрешается переходить в соседнюю свободную клетку по общей стороне.

Требуется найти длину кратчайшего пути между точками $(x_1,y_1)$ и $(x_2,y_2)$.

Ограничения:

```math
1 \leq N,M \leq 1000
```

```math
1 \leq x_1,x_2 \leq M
```

```math
1 \leq y_1,y_2 \leq N
```

**Ввод:** размеры лабиринта, его описание и координаты начальной и конечной клеток.

**Вывод:** длина кратчайшего пути или `-1`, если добраться до конечной клетки невозможно.

[Код решения](solutions/solution_C.cpp)

### D. Канат в лабиринте

Дан прямоугольный лабиринт из свободных и занятых клеток.

Из одной свободной клетки можно перейти в другую, если они имеют общую сторону. Между любыми двумя свободными клетками существует ровно один путь.

Требуется найти минимальную длину каната, которой гарантированно хватит для соединения любых двух свободных клеток.

Искомая длина равна диаметру графа свободных клеток, то есть максимальному расстоянию между двумя клетками.

Ограничения:

```math
3 \leq n,m \leq 820
```

Символ `#` обозначает занятую клетку, а `.` — свободную.

**Ввод:** размеры лабиринта и его описание.

**Вывод:** минимально необходимая длина каната.

### E. Стоимость проезда

Страна состоит из $n$ городов и $m$ односторонних дорог. Проезд по каждой дороге стоит одну золотую монету.

Город $s$ является столицей.

Требуется найти минимальную стоимость проезда из каждого города до столицы.

Ограничения:

```math
1 \leq n \leq 10^5
```

```math
0 \leq m \leq 10^5
```

Пара $(a,b)$ означает наличие дороги из города $a$ в город $b$.

**Ввод:** числа $n$, $s$ и $m$, затем описания дорог.

**Вывод:** для каждого города минимальная стоимость пути до столицы. Если пути нет, вывести `-1`.

[Код решения](solutions/solution_E.cpp)

### F. Дейкстра

Дан ориентированный взвешенный граф.

Требуется найти кратчайшее расстояние от вершины $S$ до вершины $F$.

Граф задан матрицей смежности. Значение `-1` означает отсутствие ребра, а неотрицательное число — вес ребра.

Ограничения:

```math
1 \leq N \leq 2000
```

```math
1 \leq S,F \leq N
```

Вес каждого ребра:

```math
0 \leq w \leq 10000
```

На главной диагонали матрицы записаны нули.

**Ввод:** числа $N$, $S$ и $F$, затем матрица смежности размером $N \times N$.

**Вывод:** кратчайшее расстояние от $S$ до $F$ или `-1`, если пути не существует.

[Код решения](solutions/solution_F.cpp)

### G. Алгоритм Дейкстры за M log N

Для нескольких неориентированных взвешенных графов требуется найти расстояния от заданной начальной вершины до всех остальных.

Все веса рёбер неотрицательны. Алгоритм должен эффективно работать на больших разреженных графах.

Требуемая сложность:

```math
O(M\log N)
```

Ограничения для одного графа:

```math
N \leq 60000
```

```math
M \leq 200000
```

```math
0 \leq w_i \leq 20000
```

Количество графов:

```math
NUM \leq 5
```

Вершины нумеруются от $0$ до $N-1$.

**Ввод:** число графов `NUM`. Для каждого графа задаются числа $N$ и $M$, рёбра с весами и начальная вершина.

**Вывод:** для каждого графа вывести расстояния от начальной вершины до всех вершин. Для недостижимой вершины вывести `2009000999`.

### I. Электробусы

Между корпусами ходят электробусы по заданному расписанию.

Каждый рейс содержит:

* корпус отправления;
* время отправления;
* корпус назначения;
* время прибытия.

В момент времени $0$ пассажир находится в корпусе $D$. Если он прибывает в корпус в момент $t$, то может воспользоваться любым рейсом, отправляющимся не раньше $t$.

Требуется определить минимальное время прибытия в корпус $V$.

Ограничения:

```math
1 \leq N \leq 100
```

```math
0 \leq R \leq 10^4
```

Время отправления и прибытия:

```math
0 \leq t \leq 10^4
```

**Ввод:** число корпусов $N$, начальный корпус $D$, конечный корпус $V$, количество рейсов $R$, затем расписание рейсов.

**Вывод:** минимальное время прибытия в корпус $V$ или `-1`, если добраться невозможно.

### J. Среднее расстояние

Дан ориентированный взвешенный граф, описывающий дорожную сеть.

Требуется найти среднюю длину кратчайшего пути между различными городами. Учитываются только упорядоченные пары $(a,b)$, для которых существует путь из $a$ в $b$.

Среднее расстояние вычисляется по формуле

```math
\frac{\sum l(a,b)}{\text{количество достижимых пар }(a,b)}
```

где $a \neq b$, а $l(a,b)$ — длина кратчайшего пути из города $a$ в город $b$.

Ограничения:

```math
1 \leq N \leq 100
```

```math
1 \leq K \leq N(N-1)
```

Вес дороги:

```math
1 \leq l_i \leq 1000
```

**Ввод:** числа $N$ и $K$, затем описания односторонних дорог и их длины.

**Вывод:** среднее расстояние между городами с шестью знаками после десятичной точки.

[Код решения](solutions/solution_J.cpp)
