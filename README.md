# Алгоритмы и структуры данных на C++

В репозитории собраны решения домашних заданий по курсу «Алгоритмы и структуры данных».

Основные темы:

* обход графов: DFS и BFS;
* поиск компонент связности, циклов, мостов и точек сочленения;
* кратчайшие пути: алгоритмы Флойда, Дейкстры и Беллмана — Форда;
* бинарный поиск и операции `find` / `insert`;
* минимальные остовные деревья;
* система непересекающихся множеств;
* дерево отрезков;
* динамическое программирование;
* базовые алгоритмы на графах и деревьях.

Каждая папка соответствует отдельному домашнему заданию и содержит файл `README.md` с условиями задач и папку `solutions` с решениями на Python.
