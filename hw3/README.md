## Задачи

### A. Форд — Беллман 1

Дан ориентированный взвешенный граф, в котором могут присутствовать петли и кратные рёбра. Веса рёбер могут быть отрицательными, но циклов отрицательного веса нет.

Требуется найти длины кратчайших путей от вершины $1$ до всех остальных вершин.

Ограничения:

```math
1 \leq N \leq 100
```

```math
0 \leq M \leq 10000
```

Вес каждого ребра:

```math
-100 \leq w_i \leq 100
```

**Ввод:** числа $N$ и $M$, затем $M$ троек чисел: начало ребра, конец ребра и его вес.

**Вывод:** $N$ расстояний от вершины $1$. Если вершина недостижима, вывести для неё число `30000`.

### B. Форд — Беллман 2

Дан ориентированный граф с вершинами от $1$ до $n$.

Для каждой пары вершин $i<j$ существует ребро из $i$ в $j$. Его вес вычисляется по формуле

```math
w(i,j)=(179i+719j)\bmod 1000-500
```

Требуется найти вес кратчайшего пути из вершины $1$ в вершину $n$.

Ограничение:

```math
2 \leq n \leq 13000
```

**Ввод:** одно целое число $n$.

**Вывод:** вес кратчайшего пути из вершины $1$ в вершину $n$.

### C. Алгоритм Беллмана — Форда

Дан слабо связный ориентированный взвешенный граф.

Требуется найти кратчайшие расстояния от вершины $0$ до всех остальных вершин. Граф может содержать отрицательные циклы.

Если из вершины $0$ можно попасть в отрицательный цикл, а затем из него добраться до вершины $v$, то длина пути до $v$ может быть сколь угодно маленькой.

Ограничения:

```math
2 \leq n \leq 10^4
```

```math
n-1 \leq m \leq \min\left(n(n-1),2\cdot 10^4\right)
```

Вершины нумеруются от $0$ до $n-1$.

Для каждого ребра:

```math
0 \leq u,v \leq n-1,\qquad u\neq v
```

```math
-2\cdot 10^5 \leq w \leq 10^6
```

Гарантируется, что из вершины $0$ существует путь до всех остальных вершин.

**Ввод:** числа $n$ и $m$, затем $m$ троек $u$, $v$, $w$, задающих направленное ребро из $u$ в $v$ веса $w$.

**Вывод:** для каждой вершины вывести кратчайшее расстояние от вершины $0$. Если до вершины существуют пути сколь угодно маленького веса, вывести `-inf`.

### D. Машина времени

Между населёнными пунктами выполняются рейсы на машинах времени.

Каждый рейс задаётся четырьмя числами:

* пункт отправления;
* время отправления;
* пункт назначения;
* время прибытия.

В момент времени $0$ пассажир находится в пункте $A$. Если он прибывает в некоторый пункт в момент $T$, то может воспользоваться любым рейсом, отправляющимся в момент $T$ или позднее.

Из-за путешествий во времени момент прибытия может быть меньше момента отправления.

Требуется найти минимальное время, в которое можно оказаться в пункте $B$.

Ограничения:

```math
1 \leq N \leq 1000
```

```math
0 \leq K \leq 1000
```

Для всех значений времени:

```math
|t| \leq 10^9
```

Гарантируется, что добраться из пункта $A$ в пункт $B$ возможно.

**Ввод:** число пунктов $N$, начальный и конечный пункты $A$ и $B$, количество рейсов $K$, затем описания рейсов.

**Вывод:** минимальное возможное время прибытия в пункт $B$.

### E. Цикл отрицательного веса

Дан ориентированный взвешенный граф.

Требуется определить, существует ли в нём цикл, сумма весов рёбер которого отрицательна. Если такой цикл существует, необходимо вывести его.

Ограничение:

```math
1 \leq N \leq 100
```

Граф задаётся матрицей смежности. Значение `100000` означает отсутствие ребра.

Для существующего ребра выполняется:

```math
|w_{ij}|<100000
```

**Ввод:** число $N$, затем матрица смежности размером $N\times N$.

**Вывод:** вывести `NO`, если отрицательного цикла нет. Если цикл существует, вывести `YES`, количество вершин в записи цикла и последовательность этих вершин. Первая и последняя вершины должны совпадать.

[Код решения](solutions/solution_E.cpp)

### F. Флойд — существование кратчайшего пути

Дан ориентированный взвешенный граф.

Для каждой упорядоченной пары вершин требуется определить один из трёх случаев:

* пути между вершинами не существует;
* кратчайший путь существует;
* существуют пути сколь угодно маленького веса из-за отрицательного цикла.

Ограничение:

```math
1 \leq N \leq 100
```

Граф задан матрицей смежности. Ноль означает отсутствие ребра, любое другое число — вес существующего ребра.

```math
|w_{ij}| \leq 100
```

Для каждой пары вершин $(i,j)$ требуется вывести:

* `0`, если пути из $i$ в $j$ не существует;
* `1`, если кратчайший путь существует;
* `2`, если из $i$ можно попасть в отрицательный цикл и затем добраться до $j$.

**Ввод:** число $N$, затем матрица смежности размером $N\times N$.

**Вывод:** матрица размером $N\times N$, содержащая значения `0`, `1` или `2`.
