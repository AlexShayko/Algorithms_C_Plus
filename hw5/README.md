### A. НОД на подотрезках

Дан массив из $N$ натуральных чисел.

Требуется обработать запросы вычисления наибольшего общего делителя всех элементов на отрезке от $l$ до $r$ включительно:

```math
\gcd(a_l,a_{l+1},\ldots,a_r)
```

Ограничения:

```math
1 \leq N \leq 100000
```

```math
1 \leq a_i \leq 100000
```

Количество запросов:

```math
1 \leq K \leq 30000
```

**Ввод:** число $N$, элементы массива, число запросов $K$, затем границы каждого отрезка $l$ и $r$.

**Вывод:** для каждого запроса вывести НОД элементов соответствующего отрезка. Ответы выводятся в одну строку через пробел.

### E. Билеты на электричку

Маршрут электрички проходит через $N+1$ станцию, пронумерованную от $0$ до $N$. В электричке имеется $K$ сидячих мест.

Запрос $(x,y)$ означает, что пассажир хочет проехать от станции $x$ до станции $y$.

Билет продаётся, если на каждом участке маршрута от $x$ до $y$ имеется хотя бы одно свободное место. После продажи билета количество занятых мест на всех этих участках увеличивается на единицу.

Для запроса необходимо проверить условие

```math
\max_{i=x}^{y-1} occupied_i < K
```

Если оно выполняется, на отрезке применяется обновление

```math
occupied_i := occupied_i+1
```

Ограничения:

```math
1 \leq N \leq 200000
```

```math
1 \leq K \leq 1000
```

```math
1 \leq M \leq 100000
```

Для каждого запроса:

```math
0 \leq x<y \leq N
```

**Ввод:** числа $N$, $K$ и $M$, затем $M$ запросов $(x,y)$.

**Вывод:** для каждого запроса вывести `1`, если билет был продан, и `0`, если свободных мест не осталось.

### F. Знакочередование

Дан массив

```math
a_1,a_2,\ldots,a_n
```

Требуется обрабатывать два вида операций:

* `0 i j` — присвоить элементу $a_i$ значение $j$;
* `1 l r` — найти знакочередующуюся сумму на отрезке $[l,r]$.

Знакочередующаяся сумма имеет вид

```math
a_l-a_{l+1}+a_{l+2}-a_{l+3}+\ldots\pm a_r
```

Ограничения:

```math
1 \leq n \leq 10^5
```

```math
0 \leq a_i \leq 10^4
```

Количество операций:

```math
1 \leq m \leq 10^5
```

Для операции обновления:

```math
1 \leq i \leq n,\qquad 1 \leq j \leq 10^4
```

Для запроса суммы:

```math
1 \leq l \leq r \leq n
```

**Ввод:** число $n$, элементы массива, число операций $m$, затем описания операций.

**Вывод:** для каждой операции второго типа вывести знакочередующуюся сумму на отдельной строке.

[Код решения](solutions/solution_F.cpp)
