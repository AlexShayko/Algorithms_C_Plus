## Задачи

### A. Мультиграф

Дан неориентированный невзвешенный граф, в котором могут присутствовать петли и кратные рёбра.

Требуется построить новый граф без петель и кратных рёбер так, чтобы расстояние между любыми двумя вершинами осталось таким же, как в исходном графе.

Если две вершины находятся в разных компонентах связности, расстояние между ними считается бесконечным.

Ограничения:

```math
1 \leq n,m \leq 100000
```

**Ввод:** в первой строке числа $n$ и $m$ — количество вершин и рёбер. В следующих $m$ строках записаны пары вершин, соединённых рёбрами.

**Вывод:** новый граф в таком же формате. Рёбра можно выводить в произвольном порядке.

### B. Цветной дождь

Дан неориентированный граф, заданный матрицей смежности. Каждая вершина окрашена в один из трёх цветов:

* `1` — красный;
* `2` — синий;
* `3` — зелёный.

Ребро считается плохим, если оно соединяет вершины разных цветов.

Требуется определить количество плохих рёбер.

Ограничение:

```math
0 < N \leq 100
```

**Ввод:** число $N$, затем матрица смежности размером $N \times N$. После пустой строки записаны цвета всех вершин.

**Вывод:** количество рёбер, соединяющих вершины разных цветов.

[Код решения](solutions/solution_B.cpp)

### C. Связность

Дан неориентированный граф, в котором могут присутствовать петли и кратные рёбра.

Требуется проверить, является ли граф связным. Граф считается связным, если из любой его вершины можно попасть в любую другую.

Ограничения:

```math
1 \leq N \leq 100
```

```math
0 \leq M \leq 10000
```

**Ввод:** в первой строке числа $N$ и $M$. В следующих $M$ строках записаны концы рёбер $u_i$ и $v_i$.

**Вывод:** `YES`, если граф связный, и `NO` в противном случае.

### D. Компоненты связности

Дан неориентированный невзвешенный граф.

Требуется определить количество его компонент связности и для каждой вершины указать номер компоненты, которой она принадлежит.

Ограничения:

```math
0 < N \leq 100000
```

```math
0 \leq M \leq 100000
```

Для каждого ребра:

```math
1 \leq i,j \leq N
```

**Ввод:** в первой строке числа $N$ и $M$. В следующих $M$ строках записаны пары вершин, соединённых рёбрами.

**Вывод:** в первой строке количество компонент связности. Во второй строке — $N$ чисел, где $i$-е число является номером компоненты для вершины $i$.

Компоненты нумеруются последовательными числами, начиная с $1$.

### F. Есть ли цикл?

Дан ориентированный граф, заданный матрицей смежности.

Требуется определить, существует ли в нём ориентированный цикл.

Ограничение:

```math
N \leq 50
```

Элемент матрицы $a_{ij}$ равен $1$, если существует ребро из вершины $i$ в вершину $j$, и $0$ в противном случае.

На главной диагонали матрицы находятся нули.

**Ввод:** число $N$, затем матрица смежности размером $N \times N$.

**Вывод:** `1`, если в графе существует цикл, и `0` в противном случае.

### H. Города-1

Дана дорожная сеть, которая представляет собой дерево:

* все дороги двусторонние;
* из любого города можно добраться в любой другой;
* между любыми двумя городами существует ровно один путь.

Удалённостью города называется максимальное расстояние от него до любого другого города:

```math
e(v)=\max_{u} d(v,u)
```

Требуется найти минимальную удалённость и все города, для которых она достигается.

Ограничение:

```math
1 \leq N \leq 10^5
```

Граф содержит ровно $N-1$ дорогу.

**Ввод:** число $N$. В следующих $N-1$ строках записаны пары городов, соединённых дорогами.

**Вывод:** минимальная удалённость, количество городов с такой удалённостью и номера этих городов в возрастающем порядке.

[Код решения](solutions/solution_H.cpp)

### I. Мосты

Дан связный неориентированный граф.

Мостом называется ребро, после удаления которого граф перестаёт быть связным.

Требуется найти все мосты в графе.

Ограничения:

```math
n \leq 20000
```

```math
m \leq 200000
```

Для каждого ребра:

```math
1 \leq b_i,e_i \leq n
```

Рёбра нумеруются с $1$ в порядке их появления во входных данных.

**Ввод:** в первой строке числа $n$ и $m$. В следующих $m$ строках записаны концы рёбер $b_i$ и $e_i$.

**Вывод:** в первой строке количество мостов. Во второй строке — номера всех мостов в возрастающем порядке.
