# Домашнее задание №4

## Задачи

### A. Острова

Даны $N$ островов и план строительства $M$ двусторонних мостов. Мосты строятся строго в указанном порядке.

После постройки всех мостов гарантируется связность всех островов.

Требуется определить минимальное количество первых мостов, после строительства которых с любого острова можно добраться до любого другого.

Ограничения:

```math
1 \leq N \leq 10000
```

```math
1 \leq M \leq 50000
```

Для каждого моста:

```math
1 \leq x,y \leq N
```

**Ввод:** числа $N$ и $M$, затем $M$ пар островов, соединяемых мостами.

**Вывод:** минимальное количество построенных мостов, после которого граф станет связным.

### B. Вес компоненты

В неориентированный взвешенный граф последовательно добавляются рёбра.

Необходимо обрабатывать два вида операций:

* `1 x y w` — добавить ребро между вершинами $x$ и $y$ веса $w$;
* `2 x` — найти сумму весов всех рёбер в компоненте связности вершины $x$.

Кратные рёбра разрешены.

Ограничения:

```math
1 \leq n,m \leq 10^6
```

```math
1 \leq x<y \leq n
```

```math
1 \leq w \leq 10^3
```

**Ввод:** числа $n$ и $m$, затем $m$ операций.

**Вывод:** для каждой операции второго типа вывести сумму весов рёбер в соответствующей компоненте связности.

[Код решения](solutions/solution_B.cpp)

### C. Каркас — разминка 1

Дан массив $A$ из $N$ элементов и два индекса $q$ и $w$.

Требуется заменить все элементы массива, равные $A[q]$, на значение $A[w]$.

Ограничения:

```math
1 \leq N \leq 100
```

```math
1 \leq A_i \leq 100
```

```math
1 \leq q,w \leq N
```

Использовать дополнительный массив не требуется.

**Ввод:** число $N$, элементы массива и индексы $q$, $w$.

**Вывод:** массив после выполнения всех замен.

### D. Каркас — разминка 2

Дан полный неориентированный взвешенный граф из $N$ вершин.

В каждой вершине записано значение `0` или `1`. Гарантируется, что существует хотя бы одна вершина каждого типа.

Требуется найти две вершины, для которых:

* в первой записан `0`;
* во второй записана `1`;
* вес ребра между ними минимален.

Ограничения:

```math
2 \leq N \leq 100
```

Вес каждого ребра:

```math
1 \leq w_{ij} \leq 1000
```

**Ввод:** число $N$, матрица весов графа и значения, записанные в вершинах.

**Вывод:** номера двух подходящих вершин. Если оптимальных пар несколько, можно вывести любую.

[Код решения](solutions/solution_D.cpp)

### H. Разрезание графа

Дан неориентированный граф, над которым выполняются операции:

* `cut u v` — удалить ребро между вершинами $u$ и $v$;
* `ask u v` — проверить, находятся ли вершины $u$ и $v$ в одной компоненте связности.

Гарантируется, что каждое исходное ребро удаляется ровно один раз и после всех операций `cut` в графе не остаётся рёбер.

Ограничения:

```math
1 \leq n \leq 50000
```

```math
0 \leq m \leq 100000
```

```math
m \leq k \leq 150000
```

Граф не содержит петель и кратных рёбер.

**Ввод:** числа $n$, $m$ и $k$, затем исходные рёбра и последовательность операций.

**Вывод:** для каждой операции `ask` вывести `YES`, если вершины находятся в одной компоненте связности, и `NO` в противном случае.
